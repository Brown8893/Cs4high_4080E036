# 7-1 　什麼是輸入？ 
```
● 輸入（input）指的是進入電腦記憶體的任何資料與指令。
```
```
● 一般常用的的輸入方式包含：
  ─ 鍵盤
  ─ 指向裝置
  ─ 觸控螢幕
  ─ 筆式輸入
  ─ 體感輸入
  ─ 語音輸入
  ─ 視訊輸入
  ─ 掃描器與讀取裝置
```
# 7-2 　鍵盤
```
● 鍵盤 (keyboard) 是一種上面有許多按鍵的輸入裝置，使用者按下按鍵將資料和指令輸入電腦或行動裝置。
```
```
● 幾乎所有的鍵盤上都有……
  ─ 打字區
  ─ 功能鍵
  ─ 切換鍵
  ─ 使用者可按定位鍵
  ─ 媒體控制鈕
  ─ Internet 控制鈕
  ─ 其他特殊鍵
```
```
● 電腦及行動裝置有各式各樣的鍵盤種類可供選擇
```
```
● 人體工學鍵盤（ergonomic keyboard）的設計，可減少手腕與手部罹患重複性施力傷害（RSI）的機率

● 人體工學（ergonomics）的目標是在工作場所的設計中， 納入舒適、效率和安全等考量因素
```
# 7-3 　指向裝置
```
● 指標（pointer）是螢幕上的小型符號，它的位置與外形會隨著指向裝置的移動而改變。
```
```
● 滑鼠（mouse）是一種能夠舒適地用手掌操控的指向裝置。
  ─ 光 學滑鼠, 雷射滑鼠及觸控滑鼠
```
```
● 觸控板
  ─ 觸控板（touchpad）這種指向裝置是小型而平坦的一小塊矩形，對於壓力與動作相當敏感。
● 軌跡球
  ─ 軌跡球（trackball）屬於固定式指向裝置，在它的上方或側邊有一顆球。
```
# 7-4 　觸控螢幕
```
● 觸控螢幕（touch screen）是一種觸控感應的顯示裝置。
```
# 7-5 　筆式輸入
```
● 透過筆式輸入（pen input）方式，即可使用觸控筆（ stylus ）或數位筆（ digital pen ）在平坦的表面上書寫、繪圖或標示選取。
```
```
● 繪圖板（graphics tablet）也稱作數位板（digitizer），是一種能偵測觸控筆或數位筆的動作，並轉換成數位訊號傳送給電腦的電子塑膠板。
```
# 7-6 　 體感輸入、語音輸入及視訊輸入
```
● 體感輸入（motion input）有時也稱作姿勢辨識（gesture recognition），懸浮姿勢（air gesture）指的是在空氣中移動身體或手持輸入裝置。
```
```
● 語音輸入（voice input）是對麥克風說話來執行輸入作業的程序。

● 語音辨識（voice recognition， 也稱作speech recognition），指的是電腦或行動裝置辨識人類說話字彙的能力。
```
```
● 音訊輸入（audio input）指的是將任何聲音（如說話、音樂和音效）輸入電腦的程序。

● 音樂製作軟體（music production software） 可以讓使用者錄音、作曲、混音，以及編輯音樂和聲音。
```
```
● 視訊輸入（video input）指的是擷取動態影像，並將它們存放在電腦或行動裝置儲存媒體（如硬碟或光碟片）的程序。
  ● 數位攝影機（digital video (DV) camera）是直接將影片錄製為數位訊號
  ● 有適當連線，就能直接傳輸到電腦或行動裝置中
```
```
● 網路攝影機 (webcam) 是一種 DV 攝影機，可以用來
  ─ 可用來攝影
  ─ 或擷取靜態影像
  ─ 可將影片檔附加在電子郵件送出
  ─ 在 Internet 上發布現場影像
  ─ 視訊會議
  ─ 視訊通話
```
```
● 視訊會議（videoconference）是由位在兩個或多個不同地點的人員，透過網路或 Internet 傳送音訊與視訊而一起舉行的會議。
```
```
● 無人機（drone）：專業術語叫做無人飛行載具（unmanned aerial vehicle，UAV），是一種由包括機載電腦（onboard computer）
  加 GPS、遙控裝置，可能還有  App，全體一起共同運作的飛行器。
  
● 剛開始是軍事用途，現在還有：
  ─ 搜救、交通和氣象監測
  ─ 監視、賑災、協助農事、送貨
  ─ 空拍等
```
# 7-7 　掃描器與讀取裝置
```
● 掃描器（scanner），它是光感測輸入裝置，可讀取印刷文字和圖形，然後將結果轉換成電腦能處理的形式。
  ─ 平台式掃描器（flatbed scanner） 的作業方式與影印機類似，只是它會在記憶體中建立文件的檔案，而不是影印在紙張上。
```
```
● 光學讀取裝置（optical reader）這種裝置是使用光源來讀取字元、記號和代碼，然後將它們轉換成電腦能處理的數位資料。
  ─ 光學字元辨識（OCR）
  ─ 光學記號辨識（OMR）
```
```
● 條碼讀取裝置（bar code reader）也叫做條碼掃描器（bar code scanner），是一種使用雷射光束讀取條碼的光學讀取裝置。

● QR 碼（quick response code，QR code）在垂直和水平方向都儲存資訊。
```
```
● RFID（radio frequency identification，無線射頻識別）技術是使用無線訊號，與物件、動物或人員身上的標籤連線。

● RFID 讀取裝置（RFID reader）可透過無線電波讀取標籤上的資訊。

● RFID 的用途包含：
  ─ 記錄馬拉松選手的時間
  ─ 追蹤人員、飛機行李和放錯或被偷商品的位置等
  ─ 標記並更新庫存
  ─ 測量輪胎的壓力與溫度
  ─ 圖書館的借書作業
  ─ 管制出入
  ─ 記錄車輛通過收費站的付款狀況
```
```
● 磁條卡讀取裝置 (Magstripe readers) 可讀取信用卡、集點卡、金融卡等背面的磁條。
  ─ 信用卡
  ─ 集點卡
  ─ 銀行卡
  ─ 身分證
  ─ 其他類似的卡
```
```
● MICR（magnetic-ink character recognition， 磁性墨水字元辨識）裝置能讀取以磁性墨水印刷的文字。

● MICR 讀取裝置會將 MICR 字元轉換成電腦能處理的格式。

● 銀行業幾乎是獨家使用 MICR 來處理支票。
```
```
● 資料收集裝置（data collection device）不是從來源文件讀取或掃描得到資料，而是直接從交易或事件發生的地點取得資料。
```
# 7-8 　什麼是輸出？ 
```
● 輸出（output）指的是已經處理成有用形式的資料
```
# 7-9 　顯示器
```
● 顯示裝置（display device）或簡稱顯示器（display）或螢幕，指的是能以視覺方式傳達文字、圖像與視訊資訊的輸出裝置。

● 監視器（monitor）這種顯示裝置是被包裝成獨立的周邊裝置。
  ─ 液晶顯示監視器
```
```
● 顯示器的品質主要由它的下列特性來決定：
  ─ 解析度
  ─ 回應時間
  ─ 明亮度
  ─ 點距
  ─ 對比率
```
```
● 今日的顯示器使用數位訊號來產生圖像。
● 假如要顯示最高品質的影像，顯示器應該連接到：
  ─ DVI 連接埠
  ─ HDMI 連接埠
  ─ DisplayPort 連接埠
```
```
● 家庭使用者有時候會使用數位電視（digital television，DTV）當作顯示器。

● 高畫質電視（high-definition television， HDTV）是最先進的數位電視。

● 智慧電視（Smart TV）是一種具備上網功能的 HDTV 。
```
# 7-10 　印表機
```
● 印表機（printer）這種輸出裝置會在實體媒體上（如紙張）產生文字和圖像。
● 列出一份問卷清單，可以協助你決定最適合的印表機。
```
```
● 非撞擊式印表機（nonimpact printer）不必實際接觸紙張，即可在紙上形成文字與圖形。
  ─ 噴墨印表機
  ─ 相片印表機
  ─ 雷射印表機
  ─ 多合一印表機
  ─ 3-D 印表機
  ─ 熱感式印表機
  ─ 行動印表機
  ─ 標籤印表機
  ─ 繪圖機
  ─ 大尺寸印表機
```
```
● 噴墨印表機（ink-jet printer）是一種非撞擊式印表機，透過噴灑小滴的液態墨水在紙上， 形成文字與圖像。
  ─ 彩色與黑白
  ─ 噴墨印表機的速度，是以每分鐘能列印的頁數（pages per minute，ppm）為測量標準。
```
```
● 相片印表機（photo printer）是一種彩色印表機，可產生沖洗店品質的相片

● 許多相片印表機使用的都是噴墨式技術。
```
```
● 多合一印表機（all-in-one printer）是一種看起來像台印表機或影印機，但可提供包含印表機、掃描器、影印機，也許還加上傳真機等
  多種功能的單一裝置。
  ─ 也稱作多功能印表機（multifunction printer，MFP）
```
```
● 3D 印表機（3-D printer）使用一種被稱為「疊層製造技術」（additive manufacturing）的程序來製作出物件；做法是在立體物件
  上逐層添加物質，以水平方式一次一層。
```
```
● 熱感式印表機（thermal printer）是推動電熱式的針頭接觸感熱紙而產生影像。
  ─ 熱昇華印表機
```
```
● 行動印表機（mobile printer）是一種體積小、輕巧且使用電池為電源的印表機。
```
```
● 標籤印表機（label printer）是一種專門列印在貼紙材質上的小型印表機，列印好的標籤可以貼在各種東西上。
```
```
● 繪圖機（plotter）是用來列印高品質繪圖的精密印表機。

● 大尺寸印表機（large-format printer）可產生相片品質的彩色列印輸出。
```
```
● 撞擊式印表機（impact printer）是以機械裝置對準與紙張接觸的墨水色帶撞擊，而產生字元與圖形。
```
# 7-11 　其他輸出裝置
```
● 許多使用者會替電腦、遊戲控制器和行動裝置，自行加裝具有環場音效的喇叭（speaker）或喇叭系統，以產生高品質的聲音，
  增加玩遊戲、多媒體展示、聽音樂或觀賞影片的臨場感。
```
```
● 耳罩式耳機（headphone） 是將兩耳外圍都包含在耳機內，或是直接貼在兩耳外。
● 耳塞式耳機（earbud）則是塞在耳朵裡的耳機。
```
```
● 資料投影機（data projector）是將電腦或行動裝置螢幕上顯示的文字和影像，投影到大螢幕讓所有觀眾能清楚看見影像的裝置。
```
```
● 互動式白板（interactive whiteboard）是一種類似一般白板的觸控感應裝置，上面顯示與 它連接的電腦螢幕上的畫面，通常是透過投影機。
```
```
● 搖桿、轉盤、遊戲手把（gamepad）及動作偵測遊戲控制器，都是用來控制電腦遊戲、模擬遊戲和電玩中，玩家或物件的移動或動作的輸入裝置。
  這些裝置內建「動力回  饋」（force feedback）功能
  ─ 這是一種回應使用者動作將阻力傳送回裝置的技術
```
