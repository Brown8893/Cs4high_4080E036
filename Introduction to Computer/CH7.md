# 7-1 　什麼是輸入？ 
```
● 輸入（input）指的是進入電腦記憶體的任何資料與指令。
```
```
一般常用的的輸入方式包含：
  ─ 鍵盤
  ─ 指向裝置
  ─ 觸控螢幕
  ─ 筆式輸入
  ─ 體感輸入
  ─ 語音輸入
  ─ 視訊輸入
  ─ 掃描器與讀取裝置
