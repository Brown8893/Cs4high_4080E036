# CH6
# 學習目標
```
1. 定義作業系統的目的

2. 描述電腦和行動裝置的開機程序和關機選項

3. 解釋作業系統如何提供以下這些功能：提供使用者介面、管理程式、管理記憶體，以及協調工作

4. 說明使用者如何透過作業系統設定裝置、建立 Internet 連線，以及監控效能

5. 瞭解作業系統內建的檔案管理功能和其他工具，以及更新作業系統軟體的幾種方式

6. 說明使用者如何透過作業系統控制網路或管理資訊安全

7. 摘要說明幾種單機作業系統的功能

8. 簡要描述各種不同的伺服器作業系統

9. 摘要說明幾種行動裝置作業系統的功能
```
# 6-1 　作業系統
```
● 作業系統（operating system，OS）是一組程式，負責協調電腦或行動裝置硬體間所有的活動。
  ─ 電腦或行動裝置的開機和關機
  ─ 提供使用
  ─ 者介面
  ─ 管理程式
  ─ 管理記憶體
  ─ 協調工作
  ─ 設定裝置
  ─ 監控效能
  ─ 建立 Internet  連線
  ─ 提供管理檔案和其他裝置或媒體的相關程式
  ─ 更新作業系統軟體
  ─ 控制網路
  ─ 管理安全性
