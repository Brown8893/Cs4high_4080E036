# 6-1 　作業系統
```
● 作業系統（operating system，OS）是一組程式，負責協調電腦或行動裝置硬體間所有的活動。
  ─ 電腦或行動裝置的開機和關機
  ─ 提供使用
  ─ 者介面
  ─ 管理程式
  ─ 管理記憶體
  ─ 協調工作
  ─ 設定裝置
  ─ 監控效能
  ─ 建立 Internet  連線
  ─ 提供管理檔案和其他裝置或媒體的相關程式
  ─ 更新作業系統軟體
  ─ 控制網路
  ─ 管理安全性
```
# 6-2 　作業系統的功能
```
● 開機電腦和行動裝置
  ─ 在電腦或行動裝置關機時，使用者按電源鈕便能將它開機。
```
```
● 電源選項中包括關機（關閉電源）、進入睡眠模式和進入休眠模式。
  ● 睡眠模式（sleep mode）是把開啟中的文件和執行中的程式或 App 儲存在 RAM 中，關閉所有不需要的功能，並將電腦切換至
    低功率狀態。
    
  ● 休眠模式（hibernate mode）是先把開啟中的文件和執行中的程式或 App 儲存在內部硬碟，然後再關閉電源。
```
```
● 使用者介面（user interface，UI）負責控制你如何輸入資料和指令，以及資訊會如何呈現在螢幕上。

● 圖形化使用者介面（graphical user interface，GUI）互動，它讓使用者能透過觸碰、指向或點選功能表或視覺圖像（如按鈕）
  來下達命令。
```
```
● 命令列介面（command-line interface）是鍵入由簡短關鍵字或縮寫來表示的命令字串，或按鍵盤上的特殊鍵（如功能鍵或組合鍵） 
  來輸入資料和指令。
```
```
● 作業系統如何處理程式，將直接影響你的工作效率。作業系統可以是單工或多工執行。
  ● 單工及多工
  ● 前景及背景模式
  ● 單一使用者及多人
```
```
● 記憶體管理（memory management）功能的目的，是讓電腦或裝置的內部記憶體（也就是RAM）的使用最佳化（optimize）。

● 使用虛擬記憶體（virtual memory）技術的作業系統，會配置儲存媒體的一部分空間，當作額外的 RAM 來使用。
```
```
● 作業系統負責決定任務（task）的處理順序。
```
```
● 驅動程式（driver）是裝置驅動程式（device driver）的縮寫，它是一種讓作業系統知道如何與某特定裝置溝通的小型程式。

● 隨插即用（Plug and Play）的意思是，作業系統在你安裝或連線新裝置時能自動設定它。
```
```
● 效能監視器（performance monitor）是指負責評估並回報各種電腦資源與裝置相關資訊的程式。
```
```
● 作業系統通常會提供建立 Internet 連線的途徑。
```
```
● 很多程式，包括作業系統，有內建自動更新（automatic update）功能，會定期提供該程式的新功能或修正。
```
```
● 作業系統通常也提供使用者與管理電腦、它的裝置或它的程式相關的各式工具。
  ─ 檔案管理員
  ─ 搜尋工具
  ─ 看圖軟體
  ─ 移除安裝程式
  ─ 磁碟清理
  ─ 磁碟重組
  ─ 螢幕保護
  ─ 檔案壓縮
  ─ PC 維護
  ─ 備份與還原
  ─ 電源管理
```
```
● 有些作業系統是專門為擔任網路上的伺服器而設計的

● 多人作業系統能讓多位使用者共用印表機、Internet 連線、檔案和程式

● 網路管理人員（network administrator）使用伺服器作業系統做：
  ─ 在網路上新增或移除使用者、電腦和其他裝置。
  ─ 網管人員也使用網路上的作業系統來設定網路、安裝軟體和管理網路安全功能。
```
```
● 每個使用者建立使用者帳戶，讓使用者能登入（sign in）或存取網路或電腦上的資源。
  ─ 使用者名稱（user name）或稱使用者 ID 在同一系統中是唯一的，用來辨識個別使用者。
  ─ 密碼（password）是一組由字元組成的保密字串，與使用者名稱搭配在一起輸入，才能存取特定電腦、行動裝置或網路資源
```
# 6-4 　單機作業系統
```
● 單機作業系統（desktop operating system 或 stand-alone operating system），是一種能在桌上型電腦、筆記型電腦
  和某些平板裝置上獨立運作的完整作業系統。
  ─ Windows
  ─ macOS
  ─ UNIX
  ─ Linux
  ─ Chrome OS
```
```
● Windows 除了基本功能外，還提供以下功能：
  ─ 使用動態磚（tile）可存取 App 
  ─ 內含桌面介面
  ─ 支援觸控、滑鼠和鍵盤輸入方式
  ─ 內建電子郵件 App、行事曆 App 和瀏覽器（Edge） 
  ─ 相片、檔案和設定值都能與 Microsoft 的雲端伺服器 One Drive 同步
  ─ 內建防毒軟體、防火牆和自動更新功能來加強安全性
  ─ 在 Windows Store 提供更多應用程式可另購
```
```
● Macintosh 作業系統以易學易用而著名
  ─ macOS
```
```
● UNIX（發音如YOU-nix）是在 1970 年代
  早期開發的一種多工作業系統

● Linux 是一種很受歡迎且以 UNIX 為基礎的多工作業系統
```
```
● Chrome OS 由 Google 公司所生產，是一種以 Linux 為基礎的作業系統，專為主要執行 web app 而設計
```

# 6-6 　行動裝置作業系統
```
● 行動裝置和許多消費性電子產品的作業系統，被稱為行動裝置作業系統（mobile operating system），它常駐在韌體中。
  ─ Android
  ─ iOS
  ─ Windows (行動裝置版)
```
```
● Android 是一種由 Google 為智慧型手機和平板電腦設計的開放原始碼行動裝置作業系統，它是以 Linux 為基礎。
```
```
● iOS 是 Apple 公司專為自己公司的行動裝置設計的專屬式行動裝置作業系統。
```
```
● Windows 行動裝置版（ Windows (Mobile Edition)）由 Microsoft 公司開發，是一種在智慧型手機上執行的專屬式行動裝置作業系統
```
