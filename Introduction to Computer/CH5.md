# 5-1 　程式與 App 
```
● 程式（program）或稱為軟體（software），是由一連串相關指令所組成；這些指令是因為共同的目的而組織起來，用來告訴電腦該做什麼，以及該怎麼做。

● 應用程式（application）或簡稱 App，有時也叫做應用軟體（application software），是由一組能讓使用者更有生產力，
  或協助處理個人事務的程式集合而成。

● 作業系統（operating system）是一組用來協調電腦或行動裝置硬體之間所有活動的程式。
  ─ 工具或公用程式
  ─ 系統軟體
